# cterm
Library for ncurses-like TUI (Terminal User Interface).  
Current features:  
 - Panels
 - Widgets
 - Layouts
 - Nice API

## How to build
Prerequisites: `gcc` or `clang`, `make`
Steps:  
 - Clone the repo
 - `cd cterm`
 - `make`
 - `make test`
